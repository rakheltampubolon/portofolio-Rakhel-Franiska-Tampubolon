Sistem Absensi SMP Negeri 2 Porsea Berbasis Web dan Mobile

Sistem pencatatan kehadiran terintegrasi lintas platform yang menggabungkan aplikasi mobile untuk guru/staf dengan dashboard administrasi web untuk pengelolaan data presensi instansi secara real-time.

Project Overview

Proyek ini berfokus pada pembaruan sistem pencatatan kehadiran di lingkungan SMP Negeri 2 Porsea dengan tujuan menekan manipulasi data absensi serta mempercepat pelaporan. Arsitektur sistem dirancang agar terhubung secara sinkron antara aplikasi mobile di sisi pengguna dan web dashboard di sisi manajemen sekolah.

Berperan sebagai Business Analyst & Software Quality Assurance (SQA), tanggung jawab utama saya adalah memetakan analisis alur bisnis presensi sekolah dan merancang mekanisme integrasi pertukaran data lintas platform antara aplikasi mobile dan web agar berjalan selaras.

Saya menyusun dokumentasi kebutuhan teknis berupa spesifikasi fungsional (SRS) serta user stories terstruktur untuk tim pengembang. Dari aspek penjaminan kualitas, saya memimpin aktivitas pengujian terintegrasi (integration testing) guna memastikan sinkronisasi data ke database berjalan akurat, merancang skenario uji (test cases) menyeluruh, serta memantau dan melaporkan status perbaikan bug hingga sistem siap diimplementasikan dengan andal.
