Project Overview

Saya dipercaya memegang peran sebagai Business Analyst & Software Quality Assurance. Tugas awal saya difokuskan pada analisis kebutuhan manajemen data sekolah untuk ditransformasikan menjadi representasi sistem rancangan perangkat lunak.

Saya menyusun dokumen spesifikasi kebutuhan sistem secara formal, termasuk memodelkan interaksi sistem menggunakan Use Case Diagram dan aliran data dengan Data Flow Diagram (DFD) sebagai fondasi teknis bagi tim pengembang. Di sisi kualitas, saya merancang skenario uji fungsional yang detail guna mendeteksi inkonsistensi logika atau celah bug pada fitur manajemen guru, staf, dan siswa sebelum sistem diimplementasikan secara resmi di sekolah.
